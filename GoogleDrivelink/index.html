Instruções

Siga as instruções abaixo para conseguir o seu link para download direto.

  1. Abra o Google Drive

  2. Clique com o botão direito em um arquivo.

  3. Selecione "Gerar link" e logo em seguida "Copiar link" (deixe o link visível para todos na aba Acesso Geral).

  4. Cole o link e clique no botão "Gerar Link Direto"
